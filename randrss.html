RandRSS

RandRSS is a fun simple project I made to start off 2019. You feed it an RSS link and it returns a random title in the feed from a post. You can set it to display only the latest post, or to feature the full post. It is a pretty simple web app that was made for a simple purpose which was to solve a problem I had. I wanted to bring up a random post from the subreddit Programming Challenges and Daily Programmer. That's pretty much it. :P
